# AION2 Dashboard v2.2.4

## 发布定位

- `v2.2.4` 是基于 `v2.2.3` 的新赛季规则热修补丁。
- 重点是同步 AION2 新赛季任务/商店规则，删除过期内容，并保持当前极简工作台 UI 不变。

## 本次更新

### 1. 新赛季规则同步

- 删除 `讨伐战` 全部相关逻辑和界面内容。
- `每日副本` 改为按伺服器共享每周 `14` 次。
- `每周指令书 / 深渊下层指令 / 深渊中层指令` 保持原总次数不变，但从“每角色可买/可做”改为“伺服器共享次数池”。
- `征服 / 超越` 的奖励补充次数更新为：
  - 征服：`14`
  - 超越：`7`
- `征服 / 超越` 收益阶段阈值同步为新赛季分段。

### 2. 微风商店与奥德兑换重做

- 移除旧的“基础/额外角色”资格模型。
- 改为同伺服器统一上限模型。
- 新增跟踪提醒：
  - `远征/超越挑战券选择箱(刻印)`
  - `立即完成券: 恶梦(刻印)`
  - `深渊重镇补充券(刻印)`

### 3. 数据兼容

- 旧存档会自动迁移到新的账号级共享池结构。
- 无需手动清档。

## 校验

- `npm run typecheck`
- `npm run test:unit`
- `npm run build`
- `npm run test:smoke`
