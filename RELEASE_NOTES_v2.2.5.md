# v2.2.5 发布说明

## 版本名

`圣域规则热修 + UI 收尾`

## 本次重点

- UI 收尾：
  - 角色操作页继续做了一轮精品化收口，顶部摘要、资源区和任务区块节奏更统一。
  - 做装模拟页完成一轮排版与指标卡精修。
- 商店文案修正：
  - `未知键队挑战券(刻印)` 更正为 `未知缝隙挑战券(刻印)`。
  - `深渊重镇补充券(刻印)` 更正为 `深渊重铸补充券(刻印)`。
- 圣域规则热修：
  - `圣域：卢德莱` 恢复为每周 `挑战 4 次 / 开箱 2 次`。
  - `圣域：侵蚀净化所` 恢复为每周 `挑战 4 次 / 开箱 2 次`。
  - 每次开箱默认消耗 `80` 奥德。
  - `深渊重铸补充券(刻印)` 现在明确对应 `圣域：卢德莱`，并提供 `+1 挑战 / +1 开箱`。
- 兼容处理：
  - 旧存档会自动迁移到新的圣域挑战/开箱双计数模型。

## 校验

- `npm run typecheck`
- `npm run test:unit`
- `npm run test:smoke`
- `npm run dist:win`
- `npm run check:release-yml`
