# AION2 Dashboard v2.2.2

发布日期：2026-03-12

版本名：极简工作台版补丁 2

## 本次版本定位

- `v2.2.2` 是基于 `v2.2.1` 的做装模拟器增强补丁。
- 不扩张系统边界，重点把“先录材料、再看缺口、最后换算成本”的工作流补完整。

## 功能与修复

- 做装模拟器结果区顺序优化：
  - `材料明细与库存修正`
  - `材料缺口与采购清单`
  - `模拟结论`
  - `金价换算辅助`
- 新增 `材料缺口与采购清单`：
  - 汇总显示 `缺料种类 / 总缺口数量 / 预计采购成本`
  - 仅列出当前真正缺料的材料
  - 支持一键复制采购清单
- 新增 `金价换算辅助`：
  - 支持 `1元=XX万基纳`
  - 支持 `1万基纳=XX元`
  - 支持交易行税后修正
  - 可直接估算补齐材料缺口大约还要收多少 `RMB`
