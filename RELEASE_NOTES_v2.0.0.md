# AION2 Dashboard v2.0.0

发布日期：2026-02-18

## 2.0 版本重点

- 工坊模块 2.0 主链路正式上线：
  - `OCR抓价器`
  - `市场分析器`
  - `周期性波动提示`
  - `做装模拟器`
  - `库存管理`
  - `材料逆向推导制造推荐工具`
- OCR 识别迁移为 ONNX 本地推理（免 Python / 免 PaddleOCR 手动部署）。
- 交易行支持双市场价格导入（伺服器 / 世界），并支持双曲线对比。
- 名称纠错能力增强（简繁转换、限定词修复、领域词组归一化）。
- 价格风控升级为“历史偏离 + 分类绝对阈值”并行，降低 OCR 误价写入风险。

## 角色与总览体验增强

- 角色总览支持拖拽排序。
- 快速录入新增回廊（下层/中层）录入。
- 待办优先级支持用户自定义偏好。
- 角色卡补充职业与装分信息展示。
- 倒计时与恢复规则已按当前机制校准。

## 关键规则（当前实现）

- 每日重置：`05:00`
- 每周重置：`周三 05:00`
- 奥德基础能量恢复：`02:00 / 05:00 / 08:00 / 11:00 / 14:00 / 17:00 / 20:00 / 23:00`（每次 `+15`）
- 远征恢复：`05:00 / 13:00 / 21:00`
- 超越恢复：`05:00 / 17:00`
- 回廊刷新：`周二 / 周四 / 周六 21:00`

## 工坊 OCR 与价格导入修复

- 双列价格导入与列角色识别稳定性提升。
- 常见误识别词组增加归一化修复（含简繁混写场景）。
- 对 `閃耀的...` 前缀道具增加忽略规则，避免误价污染。
- 更名修正：
  - `達人閃耀的法珠` -> `達人閃耀的真實法珠`
  - 内置目录与历史映射已同步兼容。
