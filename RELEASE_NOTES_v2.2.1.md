# AION2 Dashboard v2.2.1

发布日期：2026-03-12

版本名：极简工作台版补丁 1

## 本次版本定位

- `v2.2.1` 是基于 `v2.2.0` 的规则热修补丁。
- 不改整体 UI 方向，主要修正圣域周本规则变化，并补充一份可复用的 UI 架构说明文档。

## 功能与修复

- 圣域规则更新：
  - 旧规则：`挑战 4 次 + 开箱 2 次`
  - 新规则：两个独立圣域周本，各自 `2` 次
- 角色操作页改为两张独立圣域卡：
  - `圣域：卢德莱`
  - `圣域：侵蚀净化所`
- 两张圣域卡的逻辑一致：
  - 都只保留 `圣域完成`
  - 每次默认扣除 `80` 奥德
- 总览与优先级待办同步改为新的双圣域语义：
  - 不再显示旧的“挑战 / 开箱”组合说明
- 手动设定同步更新：
  - 改为分别录入 `卢德莱` 和 `侵蚀净化所` 的剩余次数
  - 两者上限都为 `2`

## 文档

- 新增 UI 风格复用说明：
  - `UI_STYLE_ARCHITECTURE_GUIDE.md`
  - 可用于在其他项目中复用当前这套极简工作台 UI 架构与视觉特征
