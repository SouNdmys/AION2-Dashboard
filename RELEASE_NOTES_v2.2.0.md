# AION2 Dashboard v2.2.0

发布日期：2026-03-09

版本名：极简工作台版

## 本次版本定位

- `v2.2.0` 是当前阶段的稳定收口版。
- 重点完成 Dashboard 与做装模拟的极简化改造，并修复一批影响录入准确性的逻辑问题。

## 体验与界面

- 角色总览与角色操作整体改版：
  - 总览卡恢复轻语义色分组，重点项更容易扫视。
  - 角色操作页压缩顶部与任务卡密度，主任务更突出。
  - 资源区重排为更稳定的工具带结构，减少空白浪费。
- 左侧栏收敛为目录式导航：
  - 支持收起，整体更接近单任务工作台。
- “工坊”正式更名为“做装模拟”：
  - 主视图默认只保留做装模拟器。
  - `OCR / 市场分析 / 历史价格 / 库存` 后撤到“市场工具”。
- 专业工具区改为按卡片单独折叠：
  - 不再一次性展开全部内容。

## 逻辑修复

- 回廊同步逻辑修复：
  - “同步回廊”现在表示同步当前账号角色的回廊上限。
  - 例如中层同步为 `2` 后，相关视图会显示为 `2/2`，不再错误显示为 `2/3`。
- 快捷录入修复：
  - `每日使命` 默认按增量录入，不再因为再次录入而把剩余次数重置错乱。
  - 仍保留“直接设总完成”用于绝对录入场景。
- 周常分类修正：
  - `每周指令书`
  - `深渊下层指令`
  - `深渊中层指令`
  - 现在统一归入 `周常`。

## 工坊与市场工具

- 市场分析器首屏恢复明确的“查看曲线”入口。
- 市场工具内各卡片支持独立折叠，按需展开。
- 做装模拟器继续维持单任务主视图，减少抓价与管理信息对主流程的干扰。
