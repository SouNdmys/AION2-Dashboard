# AION2 Dashboard v2.2.3

发布日期：2026-03-17

版本名：极简工作台版补丁 3

## 本次版本定位

- `v2.2.3` 是基于 `v2.2.2` 的更新体验补丁。
- 重点不是新功能，而是把安装版更新流程改成更接近“打开即更新”的体验。

## 功能与修复

- 启动即更新：
  - Windows 安装版启动时会先检查更新
  - 若发现新版本，主界面先不进入
  - 启动页会显示 `正在更新到 vX.Y.Z...`
  - 更新包下载完成后静默安装，并自动重启进入新版
- 更新失败回退：
  - 若更新检查或下载失败，不会锁死应用
  - 程序会继续进入当前旧版，保证可用性
- 自动化回归适配：
  - `smoke:ui`
  - `ui-baseline-phase0-capture`
  - 两者都已兼容新的启动更新屏

## 回归验证

- 已执行 `npm run typecheck`
- 已执行 `npm run test:smoke`
- 已执行 `npm run dist:win`
- 已执行 `npm run check:release-yml`
