# AION2 Dashboard v2.1.4

发布日期：2026-03-09

## 本次版本定位

- `v2.1.4` 是一版命名与体验收口补丁。
- 不改核心业务逻辑，重点是让工坊入口和工具分组更贴近真实使用场景。

## 功能与修复

- 工坊更名为“做装模拟”：
  - 顶部导航与左侧导航统一使用 `做装模拟` 名称。
  - 加载中、初始化失败、操作失败等兜底文案同步收口。
- “专业工具”更名为“市场工具”：
  - 更准确表达该区域承载的抓价、曲线、历史价格与库存辅助能力。
- 测试链路同步：
  - 冒烟测试更新为新导航名称。
  - 基线截图脚本更新为新折叠区名称，避免 UI 文案调整导致测试误报。

## 回归验证

- 已执行 `npm run typecheck`
- 已执行 `npm run build`
- 已执行 `npm run test:smoke`
